# Brand Guidelines

## Overview

Apply Anthropic brand colors and typography to artifacts that should visually align with Anthropic styling.

## Rules

- Use the official Anthropic palette and typography consistently.
- Keep the result readable first; brand treatment should not reduce clarity.
- Prefer subtle, coherent brand application over decorative overuse.
- Fall back gracefully when the preferred fonts are unavailable.

## When to use

- branded visual styling
- Anthropic color application
- typography choices for Anthropic-like artifacts

## Workflow

1. Identify which parts of the artifact need brand treatment.
2. Apply the core palette and type hierarchy.
3. Keep accents restrained and on-brand.
4. Check contrast, readability, and visual consistency.

## Brand core

- dark: `#141413`
- light: `#faf9f5`
- gray: `#b0aea5`, `#e8e6dc`
- accents: `#d97757`, `#6a9bcc`, `#788c5d`
- headings: Poppins
- body: Lora
