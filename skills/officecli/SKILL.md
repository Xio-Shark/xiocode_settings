# officecli

AI-friendly CLI for .docx, .xlsx, .pptx. Single binary, no dependencies, no Office installation needed.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/iOfficeAI/OfficeCLI/main/install.sh | bash
```

## Usage

```bash
officecli inspect file.docx          # Inspect document structure
officecli create report.docx         # Create new document
officecli modify file.docx           # Modify document
officecli analyze data.xlsx          # Analyze spreadsheet
```
